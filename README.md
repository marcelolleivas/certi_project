## Setup

1 - Create a Virtualenv
```
python3.7 -m venv venv
```
2 - Activate the venv
```
source ./venv/bin/activate
```
3 - Install requirements

at the project root directory
```
pip install -r requirements.txt
```
## How to run
At the root directory
```
robot -d ./results src/tests
```
